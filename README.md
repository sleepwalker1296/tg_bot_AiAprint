# AiAprint — Telegram Bot для принтов на футболках

Бот для создания эксклюзивных принтов на футболках с автомобилями. Клиент отправляет фото авто, ИИ генерирует дизайн принта, клиент получает превью с водяным знаком, администратор — оригинал в высоком качестве.

## Возможности

- 📸 Приём фото от пользователя
- 🤖 AI-генерация дизайна принта (OpenAI DALL-E 3 / Stability AI / Replicate)
- 🖼 Предпросмотр с водяным знаком и размытием для клиента
- 📁 Отправка HQ-файла без водяного знака администратору
- 📋 Интеграция с МойСклад (создание заказов покупателей)
- 🗄 SQLite база данных для хранения заказов
- 🔔 Уведомления администратору о новых заказах и подтверждениях

## Установка

```bash
# Клонировать репозиторий
git clone <repo>
cd tg_bot_AiAprint

# Создать виртуальное окружение
python -m venv venv
source venv/bin/activate  # Linux/Mac
# или: venv\Scripts\activate  # Windows

# Установить зависимости
pip install -r requirements.txt

# Скопировать и заполнить .env
cp .env.example .env
nano .env

# Создать логотип водяного знака (опционально)
python create_watermark.py

# Запустить бота
python bot.py
```

## Конфигурация (.env)

| Переменная | Описание |
|---|---|
| `BOT_TOKEN` | Токен Telegram бота (от @BotFather) |
| `ADMIN_IDS` | Telegram ID администраторов через запятую |
| `AI_PROVIDER` | Провайдер ИИ: `openai`, `stability`, `replicate` |
| `OPENAI_API_KEY` | Ключ OpenAI (для DALL-E 3) |
| `MOYSKLAD_TOKEN` | Токен МойСклад API |
| `MOYSKLAD_ORGANIZATION_ID` | UUID организации в МойСклад |
| `BLUR_RADIUS` | Степень размытия превью (0–20) |
| `PREVIEW_QUALITY` | Качество JPEG превью (1–100) |
| `WATERMARK_TEXT` | Текст водяного знака |

## Команды бота

### Для пользователей
- `/start` — приветствие и инструкция
- `/help` — справка
- `/status` — статус последнего заказа

### Для администраторов
- `/admin` — меню администратора
- `/orders` — список последних заказов
- `/order <id>` — детали заказа
- `/hq <id>` — отправить HQ файл пользователю
- `/setstatus <id> <статус>` — изменить статус
- `/mscheck` — проверить подключение к МойСклад
- `/stats` — статистика

## Статусы заказов

| Статус | Описание |
|---|---|
| `pending` | Ожидает обработки |
| `generating` | Генерируется дизайн |
| `preview_sent` | Превью отправлено пользователю |
| `confirmed` | Пользователь подтвердил заказ |
| `in_production` | В производстве |
| `shipped` | Отправлен (CDEK и др.) |
| `delivered` | Доставлен |
| `cancelled` | Отменён |

## Интеграция с МойСклад

При подтверждении заказа пользователем автоматически создаётся «Заказ покупателя» в МойСклад с:
- Номером `TG-XXXXX`
- Описанием с данными пользователя Telegram
- Привязкой к организации и складу
